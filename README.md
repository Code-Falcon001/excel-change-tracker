# excel-change-tracker

 Tracker task pane page.

## GitHub Pages
This repository is intended to be published with:
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

## Entry points
- Root URL -> `index.html`
- Task pane page -> `taskpane.html`

## Notes
- `index.html` redirects to `taskpane.html`
- `taskpane.html` is the actual hosted page
- `taskpane.css` contains the styling
- `taskpane.js` contains the browser-safe logic and Office.js readiness handling
